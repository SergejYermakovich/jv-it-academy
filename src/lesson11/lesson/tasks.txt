Задания по теме "Дженерики":

1. Напишите класс, который принимает на вход массив объектов любого типа и выводит его содержимое в обратном порядке.
2. Создайте обобщенный метод, который принимает на вход два массива одного типа и возвращает новый массив, состоящий из элементов обоих массивов.
3. Напишите класс, который реализует стек с помощью обобщенного типа. Стек должен иметь методы push, pop и peek.
4. Создайте обобщенный класс, который реализует очередь с помощью двух стеков. Очередь должна иметь методы add, remove и peek.
5. Напишите метод, который принимает на вход список объектов любого типа и возвращает новый список, состоящий только из уникальных элементов.

Задания по теме "Enum":

1. Создайте перечисление с названиями месяцев года. Напишите метод, который принимает на вход номер месяца и возвращает его название.
2. Напишите перечисление с возможными цветами светофора. Напишите метод, который принимает на вход цвет и возвращает следующий цвет светофора.
3. Создайте перечисление с возможными действиями для калькулятора (сложение, вычитание, умножение, деление). Напишите метод, который принимает на вход два числа и действие, и возвращает результат операции.
4. Напишите перечисление с возможными типами музыкальных инструментов (гитара, барабаны, фортепиано и т.д.). Напишите класс, который принимает на вход тип инструмента и возвращает соответствующий объект.
5. Создайте перечисление с возможными размерами одежды (S, M, L, XL). Напишите метод, который принимает на вход размер и возвращает следующий размер в порядке увеличения.
