Аннотация (Annotation) в Java - это механизм, который позволяет разработчикам добавлять метаданные к коду.
Аннотации используются для предоставления дополнительной информации о классах, методах, переменных и других элементах программы.

Аннотации в Java обозначаются символом @ перед их именем и могут быть использованы для различных целей, таких как описания поведения программы,
генерации кода, документирования и т.д.

Примеры встроенных аннотаций в Java:
- @Override: указывает компилятору, что метод переопределяет метод из суперкласса.
- @Deprecated: помечает элемент как устаревший, предупреждая разработчиков об использовании других альтернатив.
- @SuppressWarnings: подавляет предупреждения компилятора или IDE.

Разработчики также могут создавать собственные пользовательские аннотации для своих нужд, определяя их с помощью аннотации @interface.
