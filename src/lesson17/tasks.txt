2. Напишите лямбда-выражение, которое принимает список строк и возвращает список строк, длина которых больше 5 символов.
3. Напишите лямбда-выражение, которое принимает список целых чисел и возвращает список квадратов этих чисел.
4. Напишите лямбда-выражение, которое принимает два списка целых чисел и возвращает список, состоящий из сумм соответствующих элементов этих списков.
5. Напишите лямбда-выражение, которое принимает список строк и возвращает список строк, отсортированный по длине строк в порядке возрастания.
