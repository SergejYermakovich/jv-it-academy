Лямбда-выражения - это способ определения анонимных функций в функциональных языках программирования.
Они позволяют создавать функции, которые не имеют имени и могут быть переданы как аргументы другим функциям.

Синтаксис лямбда-выражения обычно выглядит так:

(параметры) -> выражение

Например, лямбда-выражение, которое принимает два параметра и возвращает их сумму, может выглядеть так:

(x, y) -> x + y

Лямбда-выражения могут использоваться вместо объявления отдельных функций.
Они могут быть переданы в качестве аргументов функций или сохранены в переменных.
Также они могут использоваться вместо анонимных классов для реализации интерфейсов.

Например, следующее лямбда-выражение может быть использовано для сортировки списка строк по длине:

list.sort((s1, s2) -> s1.length() - s2.length())

Лямбда-выражения являются одним из ключевых элементов функционального программирования и широко используются в языках программирования,
таких как Java, Python, JavaScript и других.
