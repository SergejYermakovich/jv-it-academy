Описание
Приложение «Console Shop».
Эта задача требует реализации базовых классов и функций.
Задача
1. Реализуйте следующие классы:
• Продукт, включающий все основные свойства (один класс для всех продуктов).
• Категория (одна категория — отдельный класс)
• Каталог с одним уровнем категорий.
2. Реализуйте консольное приложение, которое:
• Создает каталог
o Добавляет категории в каталог
o Добавляет товары в категории
• Распечатывает содержимое каталога.

Список категорий должен быть создан на основе списка классов в
целевой пакет. Каждый класс в этом пакете должен относиться к одной категории.
и иметь константу с именем категории. Соответственно, это должна быть особенность
добавить новую категорию, добавив новый класс в пакет и перезапустив
приложение.
Во время этой реализации следует использовать только классы JDK. Библиотеки не допускаются.
